 
# CRM System PHP 8

## Run Locally  
Clone the project  

~~~bash  
  git clone ....
~~~

Go to the project directory  

~~~bash  
  cd CRM System php 8
~~~
Install dependencies

~~~bash  
    docker build -t php-crm ./docker
    docker image ls
    docker run -d -p 80:80 php-crm
    
~~~

Start the server  

~~~bash  
npm run start
~~~
